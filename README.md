
# How to Edit

## With your favor editor and git command

	1. clone weekly repor

``` shell
git clone https://github.com/jared1980/weeklystatus.git
```

	2. Update weekly report with your favor editor.
	
	3. sync to remote

``` shell
git pull --rebase
git push origin main
```
